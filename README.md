# HMK Signup

HMK Signup is a web application for managing registrations for the **SCL Hallenmehrkampf (HMK)**.  
It provides a simple and robust signup flow for trainers and athletes, including Excel import/export and automated confirmation emails.

The application is built with **React + Vite** and is connected to **Supabase** for data storage and **Resend** for transactional emails.

---

## Features

- Trainer and athlete registration forms  
- Excel template download and Excel import
- Input validation (email, phone number, athlete data)
- Automatic confirmation emails via Resend
- Persistent storage using Supabase (PostgreSQL)
- Modern, responsive UI

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite  
- **Backend / Database:** Supabase (PostgreSQL, Auth, Edge Functions)
- **Email Service:** Resend
- **Deployment:** Vercel
